## Controls

- **W**/**S** to move forwards/backwards
- **A**/**D** to turn left/right
- **P** activates auto-pilot
- **L** drops box
- **R** reset scene

## Extra Features

- The vertical flaps move smoothly with player input
- Boxes have an opening animation when hitting the ground
